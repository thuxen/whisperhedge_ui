# Getting Started with WhisperHedge

Welcome! This guide will walk you through setting up your WhisperHedge account and adding your first LP position.

## Overview

Getting started with WhisperHedge takes just a few minutes:

1. **Create an account** - Sign up with your email (no password needed!)
2. **Verify your email** - Click the magic link we send you
3. **Add API keys** - Connect to your trading platforms
4. **Add positions** - Start tracking your LP positions
5. **Configure alerts** - Set up notifications for position health

## What You'll Need

Before you start, make sure you have:

- ✅ An email address
- ✅ Access to your Hyperliquid account
- ✅ Your Liquidity Pool NFT ID
- ✅ 5-10 minutes to complete setup

## Quick Tips

!!! tip "Start Small"
    Begin by adding just one position to familiarize yourself with the platform before adding all your positions.

!!! warning "API Key Security"
    Always use **read-only** API keys. Never give WhisperHedge trading permissions.
