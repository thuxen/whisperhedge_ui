# Getting Started with WhisperHedge

Add API keys → Add your first position → Set hedging strategy. Here's how:

## Step 1: Add Your HL API Keys
Securely add your Hyperliquid API keys to enable automated hedging. Each position requires a unique API key with read-only permissions. See our [API Keys & Authentication guide](../api-keys/index.md) for detailed instructions.

## Step 2: Add Your First Position
Follow our guide to [add your first LP position](first-position.md) and now we are almost ready to protect your position.

## Step 3: Set Hedging Strategy
Configure your automated hedging strategy to protect against impermanent loss and market movements. Choose from various strategies based on your risk tolerance and market outlook. See our [Hedging Strategy guide](../hedging-strategy/index.md) for available options.

That's it—you're hedging with WhisperHedge!
