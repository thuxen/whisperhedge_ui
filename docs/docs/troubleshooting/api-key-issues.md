# API Key Issues

Troubleshooting API key problems.

## Invalid API Key Error

**Symptoms:**
- "Invalid API Key" message
- Position won't verify
- Authentication fails

**Solutions:**
1. Copy key again (avoid extra spaces)
2. Verify key exists in platform
3. Check permissions are read-only
4. Regenerate if needed

## Permission Denied

**Symptoms:**
- "Permission Denied" error
- Can't access position data

**Solutions:**
1. Verify read permissions enabled
2. Check subaccount access
3. Regenerate with correct permissions

## Subaccount Not Found

**Symptoms:**
- "Subaccount not found" error
- Wrong data displayed

**Solutions:**
1. Verify exact subaccount name
2. Check capitalization and spaces
3. Confirm subaccount exists

## Key Rotation Issues

**Symptoms:**
- Old key still works after deletion
- New key not recognized

**Solutions:**
1. Wait 5-10 minutes for sync
2. Clear browser cache
3. Verify deletion in platform
4. Contact support if persists
