# Contact Support

Get help from the WhisperHedge team.

## Email Support

**support@whisperhedge.com**

We typically respond within:
- Critical issues: 2-4 hours
- General questions: 24 hours
- Feature requests: 48 hours

## What to Include

When contacting support, please include:

1. **Your email address** (account email)
2. **Issue description** (what's happening)
3. **Steps to reproduce** (how to trigger the issue)
4. **Screenshots** (if applicable)
5. **Browser/device** (what you're using)

## Before Contacting Support

Try these first:

- Check [FAQ](../faq.md)
- Review [Troubleshooting](index.md)
- Search documentation
- Try manual refresh
- Clear browser cache

## Emergency Issues

For critical security issues:
- Email: security@whisperhedge.com
- Include "SECURITY" in subject
- We respond within 2 hours

## Feature Requests

Have an idea? We'd love to hear it!

- Email: support@whisperhedge.com
- Subject: "Feature Request"
- Describe your use case
- Explain the benefit

## Bug Reports

Found a bug?

- Email: support@whisperhedge.com
- Subject: "Bug Report"
- Include steps to reproduce
- Add screenshots if possible
