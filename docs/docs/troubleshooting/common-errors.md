# Common Errors

Solutions to frequently encountered errors.

## "Invalid API Key"

**Cause:** API key format incorrect or permissions insufficient

**Solutions:**
- Verify key copied correctly (no spaces)
- Check read-only permissions enabled
- Regenerate key if needed
- Verify correct subaccount

## "Position Not Found"

**Cause:** Position doesn't exist or wrong credentials

**Solutions:**
- Verify NFT ID or API key is correct
- Check correct network selected
- Confirm position still exists
- Try manual refresh

## "Plan Limit Exceeded"

**Cause:** TVL or position count exceeds plan limits

**Solutions:**
- Upgrade plan
- Remove some positions
- Check current limits in dashboard

## "Permission Denied"

**Cause:** API key lacks required permissions

**Solutions:**
- Verify read permissions enabled
- Regenerate key with correct permissions
- Check subaccount access
