# Position Setup

Learn how to set up your LP positions for hedging in WhisperHedge.

## Supported Protocols

WhisperHedge currently supports:

- **Hyperliquid** - For hedging positions
- **Uniswap V3** - Concentrated liquidity positions

## What You'll Need

### For Hyperliquid

- Read-only API key
- API secret
- Subaccount name (if using subaccounts)

[Get Hyperliquid API keys →](../trading-accounts/hyperliquid-keys.md)

### For Uniswap V3

- NFT Token ID
- Wallet address holding the NFT
- Network (Ethereum, Polygon, etc.)

[Find your NFT ID →](finding-nft-id.md)

## Quick Start

1. **Add API keys** - Connect your Hyperliquid account
2. **Add position** - Enter your LP details
3. **Start hedging** - Automatic protection begins

[Add your first position →](../getting-started/index.md)
