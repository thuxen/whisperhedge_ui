# Position Setup

Learn how to set up and configure your LP positions for tracking in WhisperHedge.

## Supported Protocols

WhisperHedge currently supports:

- **Hyperliquid** - HLP (Hyperliquid Liquidity Provider) positions
- **Uniswap V3** - Concentrated liquidity positions

## What You'll Need

### For Hyperliquid Positions

- Read-only API key
- API secret
- Subaccount name (if using subaccounts)

[Get Hyperliquid API keys →](../api-keys/hyperliquid-keys.md)

### For Uniswap V3 Positions

- NFT Token ID
- Wallet address holding the NFT
- Network (Ethereum, Polygon, etc.)

[Find your NFT ID →](uniswap-v3-nft-id.md)

## Quick Start

1. **Choose your protocol** - Hyperliquid or Uniswap V3
2. **Gather credentials** - API keys or NFT ID
3. **Add position** - Use the dashboard
4. **Verify tracking** - Confirm data loads

[Add your first position →](../getting-started/first-position.md)

## Tracking Frequency

Position data updates at different intervals based on your plan:

- **Free Tier** - Every 15 minutes
- **Pro Tier** - Every 5 minutes
- **Premium Tier** - Every 1 minute

You can also manually refresh anytime.

[Learn more →](tracking-frequency.md)
