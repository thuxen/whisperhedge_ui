## Setting Up Email Notifications

1. Go to Settings
2. Navigate to Notifications section
3. Toggle "Email Notifications" on
4. Click "Send Test Email" to verify
5. Configure alert thresholds (coming soon)

## What You'll Be Notified About

### Critical Alerts
- Position health drops to critical
- IL exceeds 10%
- Value drops >20% in 24h
- Position requires immediate attention

### Warning Alerts
- Position health warning
- IL increasing significantly
- Volatility spike
- Approaching risk thresholds

### Informational
- Daily position summary (optional)
- Weekly performance report (optional)
- Plan limit warnings

## Notification Frequency

To avoid spam:
- Critical: Immediate
- Warning: Max once per hour per position
- Informational: Daily/weekly digests

### Not Receiving Emails?

- Check spam/junk folder

- Try sending test email
- Contact support if persists
